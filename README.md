# Legal Agreement Generator

A browser-based tool for generating professionally formatted legal agreements. No software installation, no logins, no data sent to any server. Fill in the fields, preview the document, and download a Word file instantly.

Built as a lightweight single-page app using plain HTML, CSS, and JavaScript.

---

## What it does

- Generates four agreement types: Mutual NDA, One-Way NDA, Partner Agreement, and Reseller Agreement
- Walks users through a guided 4-step form: agreement type, party details, terms, preview and download
- Produces a formatted `.docx` file ready to review, edit, and send for signatures
- Includes a built-in "How to Use" page with instructions for non-technical users

---

## How to use it (once deployed)

1. Open the link in any browser
2. Click **Create Agreement** in the top navigation
3. Select the agreement type that fits your situation
4. Fill in details for both parties (company names, addresses, signatories)
5. Enter the agreement terms (purpose, duration, governing law)
6. Click **Preview agreement**, review the document
7. Click **Download Word document** to save a `.docx` file

## Deploying on GitHub Pages

This takes about 5 minutes.

**Step 1:** Create a new repository on GitHub

Go to [github.com/new](https://github.com/new) and create a new repository. Name it something like `agreement-generator`. You can leave it as a public repository.

**Step 2:** Upload the file

Inside the repository, click **Add file > Upload files**. Drag in the `index.html` file from this project. Commit the changes.

**Step 3:** Enable GitHub Pages

Go to your repository **Settings > Pages**. Under "Source," select **Deploy from a branch**. Set the branch to `main` and the folder to `/ (root)`. Click Save.

**Step 4:** Share the link

Within a minute or two, your site will be live at:

```
https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
```

Copy that link and share it with your team.

## Agreement types

| Type | When to use |
|------|-------------|
| Mutual NDA | Both parties are sharing confidential information with each other |
| One-Way NDA | Only your company is disclosing; the other party agrees to protect it |
| Partner Agreement | Formalizing a business collaboration with scope, compensation, and responsibilities |
| Reseller Agreement | Authorizing a third party to resell your products or services |

---

## Important notice

All agreements generated by this tool are boilerplate templates intended as starting points. They are not a substitute for qualified legal advice. Always have an attorney review any agreement before it is signed or distributed.

---

## Technical details

- Single HTML file, no build process required
- Uses [docx.js](https://docxjs.org/) (via CDN) for Word document generation
- Uses [FileSaver.js](https://github.com/eligrey/FileSaver.js/) (via CDN) for browser-side file download
- All data processing happens locally in the browser. Nothing is transmitted to any server.
- Compatible with all modern browsers (Chrome, Edge, Firefox, Safari)

---

## Customization

To add new agreement types or modify the boilerplate language, open `index.html` in any text editor and look for the `buildText()` function. Each agreement type has its own section clearly marked with comments.

To change the styling, look for the `<style>` block near the top of the file.
